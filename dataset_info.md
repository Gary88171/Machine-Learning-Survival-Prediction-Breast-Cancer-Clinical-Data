## Description

The dataset contains clinical and demographic variables related to breast cancer patients, including tumor staging, lymph node involvement, tumor size, age, and survival status. These features allow supervised machine learning modeling for survival prediction tasks.

## Purpose in This Project

This dataset was selected to demonstrate a practical biomedical machine learning workflow, including preprocessing heterogeneous clinical variables, addressing class imbalance, and evaluating survival prediction performance.

## Data Usage Notes

The dataset is publicly available for educational and research purposes. It is not included directly in this repository to respect dataset hosting policies. Users should download the dataset from Kaggle and place it in the project directory before running the notebook.
